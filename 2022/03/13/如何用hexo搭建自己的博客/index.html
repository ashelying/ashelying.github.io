如何用hexo搭建自己的博客

一.准备工作

1.创建Github账号，新建一个仓库

创建与用户名同名的仓库，后+.github.io

2.安装git工具（后续步骤可省略）

  • 安装完git工具之后，在git bash上输入命令行，输入Github用户名和邮箱验证

    • git config –global user.name “用户名“
    • git config –global user.email “邮箱“

  • 再创建SSH，一直回车

    • ssh-keygen -t rsa -C “邮箱“
3.安装Node JS

安装Node JS和npm工具

二.创建仓库

在github上创建一个新的代码仓库用户保存我们的网页

  • Your repositories->New
  • 填写仓库名 ，（后+.github.io）
  • Create repository->creating a new file，作为网站的主页（新名字必须为index.html),填写后提交
  • 在Github中找到我们主页的地址：https://用户名/,即可访问

三. 安装Hexo

  • 安装Hexo
    • npm install -g hexo-cli
  • 查看版本
    • hexo -v
  • 创建一个项目hexo-blog并初始化
    • hexo init hexo-blog
    • cd hexo-blog
    • npm install
  • 本地启动
    • hexo g
    • hexo server
  • 浏览器访问http://localhost:4000

四.更换主题（以Next T为例）

  • 安装（输入命令）

    • cd hexo-blog
    • git clone 主题网址（https://github.com/***)

  • 使用

    • 打开 _config.yml文件，该文件为站点配置文件

    • 将主题修改为next（输入命令

      • theme：next

    • 本地启动（输入命令

      • hexo g -d
      • hexo s

五.创建文章

  • 修改hexo博客目录中的_config.yml（输入命令

    • post_asset_folder:true

  • 执行如下命令创建一篇新文章（输入命令

    • hexo new post 文章名称

  • 执行之后在source_posts目录下生成了一个md文件和一个同名的资源目录（用于存放图片)

  • 在 文章名称.md 中添加内容如下

    • -–
      title: 测试文章
      date: 2021-06-10 16:35:20
      tags:
      - 原创
      categories:
      - Java
      -–

      这是一篇测试文章

  • 本地启动（输入命令）

    • hexo g -d
    • hexo s

六.发布到GitHub Pages

方法一
  • 安装hexo-deployer-git(输入命令)
    • npm install hexo-deployer-git –save
  • 修改根目录下的-congit.yml，配置GitHub相关信息
    • deploy:
      type: git
repo: *主题网址*
branch: main
token: *ghp_3KakcaPHerunNRyMerofcFd9pblU282FSbsY*
      • token为Github的Personal access token,获取方式为Settings->Personal access tokens->Generate new token
  • 部署到Github（输入命令）
    • hexo g -d
方法二
  • 直接将public目录中的文件和目录推送至Github仓库和分支中
