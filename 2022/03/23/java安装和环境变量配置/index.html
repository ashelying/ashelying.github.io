      变量值：C:\Program Files\Java\jdk-11.0.13

    • 变量名：JAVA_HOME_15

Proxy模块介绍
Intercept模块：用户显示修改HTTP请求及相应内容，并可以将拦截的HTTP请求快速发送至其他模块
  • Forward 用于发送数据
  • Drop 将该请求包丢弃
  • Intercept is off/on 拦截开关。off时，BurpSuite会自动转发所拦截的所有请求；on时，BurpSuite会将所有拦截所有符合规则的请求
  • Action 功能菜单
http history模块：记录经过代理服务器访问的所有请求

显示的信息：Host（主机）、Method（请求方式）、URL（请求地址）、Params（参数）、Edited（编辑）、Statue（状态）、Length（响应字节长度）、MIME type（响应的MLME类型）、Extension（地址文件扩展名）、Title(页面标题)、Comment（注释）、SSL、IP（目标IP地址）、Cookies、Time（发出请求时间）、Listener port （监听端口）
